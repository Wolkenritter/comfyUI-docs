# 节点操作指南

了解如何在 ComfyUI 中添加、操作和管理节点。

## 添加节点

### 方法一：双击添加

1. 双击画布空白处
2. 输入节点名称
3. 从列表中选择节点
4. 点击或回车添加

### 方法二：拖拽添加

1. 从左侧节点面板拖拽节点
2. 放到画布上

### 方法三：右键菜单

1. 右键点击画布
2. 选择「添加节点」
3. 浏览分类选择节点

## 连接节点

### 基本连接

1. 点击输出端口（右侧圆点）
2. 拖拽到输入端口（左侧圆点）
3. 释放完成连接

### 端口颜色

| 颜色 | 类型 |
|------|------|
| 绿色 | MODEL |
| 蓝色 | CONDITIONING |
| 黄色 | LATENT |
| 白色/灰色 | IMAGE |
| 紫色 | VAE |

### 断开连接

- 右键点击连线 → 删除
- 或直接拖拽端口断开

## 删除节点

| 方法 | 操作 |
|------|------|
| 键盘 | 选中节点 → Delete |
| 右键 | 右键 → 删除 |
| 菜单 | 编辑 → 删除 |

## 复制节点

| 方法 | 操作 |
|------|------|
| 快捷键 | Ctrl + D |
| 复制粘贴 | Ctrl + C → Ctrl + V |
| Alt + 拖拽 | 按住 Alt 拖拽节点 |

## 搜索节点

双击画布后输入搜索词，支持：
- 精确名称：`CLIPTextEncode`
- 模糊匹配：`clip` → CLIPTextEncode
- 中文别名（部分节点）

## 节点状态

### 正常状态

节点标题正常显示。

### 报错状态

| 状态 | 表现 | 原因 |
|------|------|------|
| 红色边框 | 缺少输入 | 必需的端口未连接 |
| 黄色边框 | 类型不匹配 | 端口类型不兼容 |

### 查看错误

将鼠标悬停在红色端口上查看错误信息。

## 节点分组

### 创建组

1. 框选多个节点
2. 右键 → 添加组
3. 设置组名称和颜色

### 编辑组

- 双击组标题编辑名称
- 拖拽边缘调整大小
- 右键 → 解散组

## 节点笔记

1. 右键节点 → 添加笔记
2. 输入说明文字
3. 笔记折叠显示在节点旁边

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl + D | 复制节点 |
| Ctrl + C / V | 复制粘贴 |
| Delete | 删除 |
| L | 锁定节点位置 |
| H | 折叠节点 |
