# 工作流使用

了解如何在 ComfyUI 中保存、加载和管理工作流。

## 保存工作流

### 方法一：菜单保存

1. 点击菜单「文件」→「保存」
2. 选择保存位置
3. 输入文件名（自动添加 .json 扩展名）

### 方法二：快捷键

- `Ctrl + S` 保存
- `Ctrl + Shift + S` 另存为

### 保存内容

保存的工作流文件包含：
- 所有节点和连接
- 节点参数设置
- 提示词内容

## 加载工作流

### 方法一：菜单加载

1. 点击菜单「文件」→「打开」
2. 选择 .json 文件
3. 工作流加载到画布

### 方法二：拖拽加载

直接将 .json 文件拖入浏览器窗口。

### 方法三：拖入图片

将 ComfyUI 生成的图片拖入，会自动提取工作流。

## 导出为图片

### 导出工作流图片

1. 点击菜单「文件」→「导出」
2. 选择「导出工作流为 PNG」
3. 图片会包含嵌入的工作流信息

### 分享图片

分享导出的图片，别人可以直接拖入 ComfyUI 加载工作流。

## 工作流模板

### 内置模板

ComfyUI 提供多个预设模板：

| 模板 | 说明 |
|------|------|
| 基础文生图 | 最简单的生成流程 |
| 基础图生图 | 参考图生成 |
| SDXL 工作流 | SDXL 专用 |

### 访问模板

1. 点击菜单「文件」→「加载模板」
2. 选择模板应用

### 自定义节点模板

部分自定义节点会提供工作流模板。

## 工作流库

### 工作流库插件

推荐安装 [ComfyUI Workspace Manager](https://github.com/11cafe/comfyui-workspace-manager)。

功能：
- 管理工作流收藏
- 分类整理
- 快速搜索
- 缩略图预览

## 版本管理

### 保存历史

- 定期保存不同版本
- 命名格式：`工作流名_v1.json`

### 版本对比

使用 Git 或第三方工具管理工作流版本。

## 常见问题

### Q: 加载工作流报错

**A**: 可能缺少自定义节点，使用 Manager 安装缺失节点。

### Q: 工作流不兼容

**A**: 不同版本的 ComfyUI 可能不兼容，更新 ComfyUI 或节点。

### Q: 图片无法加载工作流

**A**: 确认图片是 ComfyUI 生成的，非第三方工具。
