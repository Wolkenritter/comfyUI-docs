## 工作流画布

工作流画布是 ComfyUI 的核心区域，用于构建和组织节点。

### 基本操作

#### 添加节点

1. **双击画布**: 打开节点搜索对话框
2. **右键菜单**: 右键点击画布，选择添加节点
3. **拖拽**: 从左侧面板拖拽节点到画布

#### 连接节点

1. 点击节点的输出端口（右侧圆点）
2. 拖拽到目标节点的输入端口（左侧圆点）
3. 端口颜色匹配表示类型兼容

#### 选择节点

- **单个选择**: 点击节点
- **多选**: 按住 Shift 点击多个节点
- **框选**: 在画布空白处拖拽框选
- **全选**: Ctrl+A

#### 移动与缩放

- **移动视图**: 按住空格+拖拽，或滚轮拖拽
- **缩放**: 滚轮缩放
- **适应屏幕**: 双击画布空白处

### 节点操作

| 操作 | 方法 |
|------|------|
| 移动节点 | 拖拽节点 |
| 删除节点 | 选中 + Delete 键 |
| 复制节点 | Ctrl+D 或 Ctrl+C/V |
| 注释节点 | 右键 → Add Note |

## 队列与生成面板

### 队列管理

- **Queue Size**: 设置队列中待处理任务数
- **Clear Queue**: 清空队列
- **Cancel**: 取消当前生成

### 生成控制

| 按钮 | 功能 |
|------|------|
| Queue Prompt | 将当前工作流加入队列 |
| Execute | 立即执行 |
| Interrupt | 中断当前执行 |

### 进度显示

- 显示当前进度
- 预估剩余时间
- GPU 显存使用情况

## 界面自定义

### 主题设置

ComfyUI 支持明暗主题切换：

1. 点击右上角设置图标
2. 选择 Theme 选项
3. 选择 light/dark/system

### 节点样式

#### 节点颜色

可以为不同类型的节点设置不同颜色便于识别：

1. 右键节点
2. 选择 Colors
3. 选择颜色

#### 节点大小

通过设置调整节点的显示大小：

```
设置 → Node Display → Node Size
```

### 快捷键自定义

在设置中可以自定义常用操作的快捷键：

```javascript
// 示例：自定义快捷键配置
{
  "keybinds": {
    "add_node": "Space",
    "delete": "Delete",
    "duplicate": "Ctrl+D"
  }
}
```

## 提示技巧

### 高效操作

1. **批量选择**: 使用框选快速选择多个节点
2. **快速搜索**: 双击后输入节点名称快速添加
3. **节点分组**: 使用 Group 功能整理复杂工作流
4. **工作流注释**: 使用 Note 节点添加说明

### 性能优化

1. 避免不必要地连接节点端口
2. 及时清理不需要的预览节点
3. 使用适当的批量大小
4. 定期保存工作流
