# 秋叶启动器使用指南

秋叶启动器（Aaaki）是中国玩家常用的 ComfyUI 一键启动器，提供图形化界面和便捷的管理功能。

## 特点

- 🔧 **图形化界面**：告别命令行
- 📦 **一键安装节点**：简化自定义节点安装
- 🔄 **版本管理**：轻松更新 ComfyUI
- 🎮 **预设配置**：开箱即用
- 🔒 **环境隔离**：避免依赖冲突

## 安装步骤

### 1. 下载启动器压缩包

下载后解压到指定目录。

### 2. 启动

双击启动器程序，等待启动完成。

### 3. 一键启动

点击「一键启动」按钮启动 ComfyUI。

## 功能详解

### 一键启动

自动检测 Python 环境，启动 ComfyUI 服务。

### 版本管理

查看当前版本，检查更新，一键升级。

### 扩展管理

安装、卸载、禁用自定义节点。

| 功能 | 说明 |
|------|------|
| 安装新扩展 | 搜索并安装节点 |
| 更新扩展 | 一键更新所有扩展 |
| 卸载扩展 | 移除不需要的节点 |
| 禁用扩展 | 临时禁用某节点 |

### 模型管理

查看和管理已安装的模型。

|------|----------|----------|

## 常见问题

### Q: 启动报 Python 错误

**A**: 启动器会自动配置 Python，如有问题可尝试以管理员身份运行。

### Q: 无法安装节点

**A**: 确保关闭 ComfyUI 后再进行节点操作。

### Q: 更新后无法启动

**A**: 可尝试回退到之前的版本，或重新安装启动器。
