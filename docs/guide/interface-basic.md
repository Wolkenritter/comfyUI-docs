# 界面指南

了解 ComfyUI 的界面布局和基本操作。

## 界面概览

```
┌─────────────────────────────────────────────────────────────┐
│  菜单栏                                                    │
├──────────┬──────────────────────────────────────────────────┤
│          │                                                  │
│  节点    │               工作流画布                        │
│  面板    │                                                  │
│          │                                                  │
│          │                                                  │
├──────────┴──────────────────────────────────────────────────┤
│  队列 & 生成面板                                            │
└─────────────────────────────────────────────────────────────┘
```

## 菜单栏

| 菜单 | 功能 |
|------|------|
| 文件 | 新建、打开、保存、导入、导出 |
| 编辑 | 撤销、重做、复制、粘贴 |
| 视图 | 缩放、平移、重置视图 |
| 队列 | 添加到队列、执行、取消 |
| 设置 | 系统设置、界面设置 |

## 工作流画布

### 基本操作

| 操作 | 方法 |
|------|------|
| 移动视图 | 空格 + 拖拽，或滚轮拖拽 |
| 缩放 | 滚轮缩放 |
| 选择节点 | 单击节点 |
| 多选 | Shift + 点击，或框选 |
| 全选 | Ctrl + A |
| 删除 | Delete 键 |

### 添加节点

1. **双击画布**：打开节点搜索对话框
2. **右键菜单**：右键点击 → 添加节点
3. **拖拽连接**：从输出端口拖拽

### 节点连接

- 从右侧输出端口拖拽到左侧输入端口
- 颜色匹配表示类型兼容
- 右键点击连线可断开

## 节点面板

### 分类

| 分类 | 节点示例 |
|------|----------|
| 加载器 | Checkpoint Loader, VAE Load |
| 条件 | CLIP Text Encode |
| 采样器 | KSampler, WAS KSampler |
| 图像 | Load Image, Save Image |
| 潜空间 | Empty Latent Image |

### 快捷搜索

双击画布后输入节点名称，支持模糊匹配。

## 队列面板

| 区域 | 功能 |
|------|------|
| Queue Size | 显示队列长度 |
| 大按钮 | 添加到队列 |
| 执行 | 立即执行 |

## 状态栏

显示当前状态：
- 模型加载状态
- GPU 使用情况
- 生成进度

## 右键菜单

| 选项 | 功能 |
|------|------|
| 添加节点 | 打开节点菜单 |
| 添加笔记 | 添加文字注释 |
| 添加组 | 创建节点组 |
| 复制 | 复制节点 |
| 删除 | 删除节点 |

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl + N | 新建工作流 |
| Ctrl + O | 打开 |
| Ctrl + S | 保存 |
| Ctrl + Z | 撤销 |
| Ctrl + Y | 重做 |
| Ctrl + D | 复制节点 |
| Delete | 删除 |
| Space + 拖拽 | 移动视图 |
| 滚轮 | 缩放 |
