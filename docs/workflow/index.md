# 工作流示例

工作流是 ComfyUI 的核心功能，掌握工作流可以大幅提升效率。

## 工作流类型

### 图像生成

| 工作流 | 说明 | 难度 |
|------|------|------|
| [文生图](./text-to-image) | 文字生成图像 | ⭐ |
| [图生图](./image-to-image) | 图像转图像 | ⭐⭐ |
| [局部重绘](./inpainting) | 修复局部区域 | ⭐⭐ |
| [扩图](./outpainting) | 扩展图像边界 | ⭐⭐ |

### 高级功能

| 工作流 | 说明 | 难度 |
|------|------|------|
| [SDXL](./sdxl) | SDXL 模型使用 | ⭐⭐ |
| [ControlNet](./controlnet) | 结构控制 | ⭐⭐⭐ |
| [视频生成](./video) | AI 视频生成 | ⭐⭐⭐ |

### 模型处理

| 工作流 | 说明 | 难度 |
|------|------|------|
| [模型合并](./model-merging) | 合并多个模型 | ⭐⭐ |
| [模型转换](./model-convert) | 模型格式转换 | ⭐ |

## 推荐工作流

### 入门必学

1. **基础文生图** - 第一个工作流
2. **图生图** - 风格转换
3. **局部重绘** - 修复细节

### 进阶必学

1. **SDXL** - 高质量生成
2. **ControlNet** - 精准控制
3. **放大** - 提升分辨率

### 高阶可选

1. **视频生成** - AI 视频
2. **模型合并** - 自定义模型
3. **自定义节点** - 扩展功能

## 工作流管理

### 保存和加载

| 操作 | 说明 |
|------|------|
| 保存 | Ctrl+S 保存当前工作流 |
| 加载 | 拖拽 .json 文件到界面 |
| 导出 | 导出为独立文件 |

### 分享工作流

```
保存格式: .json
包含: 节点、连接、参数
不包含: 模型、LoRA

分享时注意:
- 模型路径可能不同
- 提示词需要重新编辑
```

## 工作流优化

### 性能优化

| 方法 | 效果 |
|------|------|
| 减少节点 | 加快速度 |
| 合理尺寸 | 节省显存 |
| 模型选择 | 平衡质量和速度 |

### 质量优化

| 方法 | 效果 |
|------|------|
| 增加步数 | 更细腻 |
| 调整 CFG | 更符合提示 |
| 多重采样 | 更稳定 |
