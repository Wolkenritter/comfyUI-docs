# 模型合并工作流

学习在 ComfyUI 中合并多个模型。

## 模型合并基础

模型合并可以将多个模型的优势结合。

### 合并类型

| 类型 | 说明 |
|------|------|
| 权重平均 | 直接加权平均 |
| 插值 | 按比例混合 |
| 合并检查点 | 合并完整模型 |

## 合并工具

### 官方合并功能

ComfyUI 内置合并工具：

```
节点: ModelMergeSimple
     ModelMergeSDXL
```

### 合并工作流

```
┌─────────────┐    ┌─────────────┐
│ CheckpointL │    │ CheckpointL │
│ oader (A)   │    │ oader (B)   │
└──────┬──────┘    └──────┬──────┘
       │                  │
       ↓                  ↓
┌──────────────────────────────┐
│        ModelMergeSimple       │
│        (ratio: 0.5)          │
└──────────────────┬───────────┘
                   ↓
            ┌─────────────┐
            │ SaveCheckpoint │
            └─────────────┘
```

## 合并参数

### 权重比例

| 值 | 效果 |
|------|------|
| 0.0 | 完全模型 A |
| 0.5 | 各 50% |
| 1.0 | 完全模型 B |

### 推荐比例

| 目标 | 比例 |
|------|------|
| 保持 A 风格 | 0.2-0.3 |
| 平衡 | 0.4-0.6 |
| 更接近 B | 0.7-0.8 |

## SD 1.5 合并示例

### 合并两个动漫模型

```
模型 A: anime modelo (动漫风格)
模型 B: realistic model (写实风格)

比例: 0.4 (偏向动漫)

结果: 动漫风格 + 部分写实细节
```

### 合并检查点

```
节点: CheckpointLoaderSimple
连接两个模型
设置合并比例
保存新模型
```

## SDXL 合并

### SDXL 合并节点

使用专门的 SDXL 合并节点：

```
节点: ModelMergeSDXL

输入:
- model_a: SDXL 模型 A
- model_b: SDXL 模型 B
- ratio: 合并比例

输出:
- merged_model: 合并后的模型
```

### SDXL 合并注意事项

| 注意事项 | 说明 |
|----------|------|
| 模型版本 | 必须都是 SDXL |
| VAE | 使用 SDXL 专用 VAE |
| LoRA | 合并后再应用 |

## 高级合并

### 多模型合并

合并多个模型：

```
模型 A (0.3) + 模型 B (0.3) + 模型 C (0.4)
= 新模型
```

### 使用算数节点

更灵活的合并方式：

```
节点: MathOperation
     Add, Subtract, Multiply

方法:
1. 加载多个模型
2. 使用 Math 节点计算权重
3. 合并结果
```

## 保存模型

### 保存合并结果

```
节点: SaveCheckpoint

设置:
- output_path: 保存路径
- filename: 文件名

保存位置:
ComfyUI/models/checkpoints/
```

## 合并策略

### 风格融合

| 策略 | 说明 |
|------|------|
| 动漫+写实 | 平衡风格 |
| 写实+艺术 | 增强表现力 |

### 质量提升

| 策略 | 说明 |
|------|------|
| 好模型 + 好模型 | 质量叠加 |
| 好模型 + 差模型 | 改善差模型 |
| 特定风格 | 定制属性 |

## 测试合并结果


```
原始比例: 0.5

找到最佳比例
```

## 常见问题

### Q: 合并后模型不能用

**A**:
- 检查模型版本是否兼容
- 确保格式正确（safetensors）
- 重启 ComfyUI

### Q: 合并比例怎么选

**A**:
- 从 0.5 开始测试
- 根据效果调整
- 记录满意的比例

### Q: 能否撤销合并

**A**: 不能撤销，建议保存多个版本。
