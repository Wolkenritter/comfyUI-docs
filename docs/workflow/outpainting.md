# 扩图工作流

扩展图像边界，增加画面内容。

## 扩图概念

扩图（Outpainting）是扩展图像边界的技术，可以：

- 增加画幅
- 添加元素
- 补充背景
- 改变比例

## 工作流结构

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ LoadImage   │ →  │ ImageCrop   │ →  │ EmptyLatent │
│             │    │ (保留区域)   │    │             │
└─────────────┘    └─────────────┘    └─────────────┘
                                              ↓
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ CLIPTextEn  │ →  │ KSampler    │ ←  │ CheckpointL │
│ code        │    │             │    │ oader       │
└─────────────┘    └──────┬──────┘    └─────────────┘
                          ↓
                    ┌─────────────┐    ┌─────────────┐
                    │ VAEDecode   │ →  │ ImagePaste │
                    │             │    │ (合并)     │
                    └─────────────┘    └─────────────┘
```

## 扩图方法

### 方法一：Latent 扩图

```
1. 加载原图
2. 扩大 EmptyLatentImage 尺寸
3. VAE 编码原图到 latent
4. 采样时扩展空白区域
5. 解码合并
```

### 方法二：图像拼接

```
1. 裁剪要保留的部分
2. 创建扩展区域
3. 拼接后整体处理
```

### 方法三：分步扩展

```
1. 扩展一边
2. 合并结果
3. 继续扩展另一边
4. 迭代完成
```

## 具体步骤

### 第一步：准备图像

```
加载图像: 512x512
目标尺寸: 1024x768 (扩展高度和宽度)
```

### 第二步：创建扩展 Latent

```
原尺寸: 512x512
新尺寸: 1024x768

Offset: 保持原图位置
(256, 128) - 居中偏移
```

### 第三步：编码原图

```
VAEEncode:
- 输入: 原图
- 输出: latent（原图部分）
- 扩展区域: 随机噪声
```

### 第四步：采样生成

```
提示词: 延续原图风格，描述扩展内容
denoise: 0.6-0.8

例如:
- 原图是风景 → 扩展更多天空和草地
- 原图是人物 → 扩展更多背景环境
```

### 第五步：合并输出

```
ImageComposite 或 Paste
- 保留原图区域
- 扩展区域使用生成内容
```

## 应用场景

### 增加画幅

```
原图: 512x512
目标: 1024x768

扩展: 上下各增加 128px
```

### 改变比例

```
原图: 1:1 (方图)
目标: 16:9 (宽屏)

扩展: 左右各增加 256px
```

### 添加元素

```
原图: 人物在室内
目标: 添加窗外风景

扩展: 窗户方向
提示词: window with view, outdoor scenery, natural light
```

### 延续背景

```
原图: 建筑局部
目标: 完整建筑

扩展: 上下或左右
提示词: continue the architectural style, matching building design
```

## 提示词技巧

### 风格延续

```
"in the same style as the original"
"continuing the background"
"seamless extension"
```

### 内容描述

```
"vast sky with clouds"
"spacious outdoor landscape"
"more room environment"
```

### 光照一致

```
"matching lighting conditions"
"consistent with the scene"
"natural transition"
```

## 参数设置

### 推荐配置

| 参数 | 值 | 说明 |
|------|------|------|
| Denoise | 0.6-0.8 | 足够重绘扩展区域 |
| Steps | 25-35 | 平衡质量和速度 |
| CFG | 7-10 | 遵循提示词 |

### 边缘处理

```
Blend: 50-100px
羽化: 避免明显接缝
```

## 分步扩图

### 扩展多次

```
1. 第一次：扩展 200px
2. 合并结果
3. 第二次：再扩展 200px
4. 继续直到达到目标

优点: 更容易控制
缺点: 耗时更长
```

### 横向扩展

```
原图: [A][B][C]
扩展: [A][B][C][D]

D = 扩展内容
```

### 四向扩展

```
顺序: 上 → 下 → 左 → 右
或: 左 → 右 → 上 → 下

每次扩展后合并
```

## 常见问题

### Q: 扩展区域不自然

**A**:
- 使用羽化边缘
- 调整提示词描述
- 增加 denoise

### Q: 与原图风格不一致

**A**:
- 添加风格描述到提示词
- 使用 lower denoise
- 参考原图特征

### Q: 接缝明显

**A**:
- 扩大羽化范围
- 使用 Blend 节点
- 多次微调
