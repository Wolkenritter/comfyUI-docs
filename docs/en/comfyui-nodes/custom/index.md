## Installation

### Via ComfyUI Manager

1. Right-click → Manager
2. Click "Custom Nodes Manager"
3. Search and install

### Manual Installation

```bash
cd ComfyUI/custom_nodes
git clone <repository-url>
pip install -r requirements.txt
```

## Popular Custom Nodes

### Impact Pack

Comprehensive suite of nodes:
- Advanced sampling
- Face detail
- Image processing
- Bounding boxes

### ControlNet Nodes

Extended ControlNet functionality:
- Multi-ControlNet
- ControlNet preprocessors
- Advanced settings

### LayerStyle

Photoshop layer effects:
- Blend modes
- Drop shadows
- Glows
- Borders
