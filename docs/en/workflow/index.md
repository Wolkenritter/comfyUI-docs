# Workflow Examples

Pre-built workflows for common tasks.

## Workflow Categories

### Basic Generation

| Workflow | Description |
|----------|-------------|
| [Text-to-Image](./text-to-image) | Basic generation |
| [Image-to-Image](./image-to-image) | Transform existing |
| [Inpainting](./inpainting) | Fix specific areas |
| [Outpainting](./outpainting) | Extend boundaries |

| Workflow | Description |

## Quick Start

1. Download workflow JSON
2. Drag into ComfyUI
3. Load models
4. Generate

## Workflow Files

### File Format

```
.json - ComfyUI workflow
```

### Location

Save to:
```
ComfyUI/custom/workflows/
```

Or drag directly into ComfyUI window.

## Tips

1. **Check dependencies**: Some workflows need specific nodes
2. **Update models**: Use current model versions
3. **Adjust parameters**: Customize to your needs
