# 如何升级ComfyUI

保持 ComfyUI 更新到最新版本。

## 方法一：使用 Manager 升级

### 1. 打开 Manager

在 ComfyUI 界面点击 Manager 图标。

### 2. 检查更新

1. 点击「Update All」
2. 查看可用更新
3. 点击更新按钮

### 3. 重启

更新后点击「Restart」重启。

## 方法二：手动更新

### 1. 备份

更新前备份：
- 自定义节点
- 工作流
- 重要配置

### 2. 更新 ComfyUI 核心

```bash
cd ComfyUI
git pull
```

### 3. 更新自定义节点

```bash
cd custom_nodes/xxx
git pull
```

### 4. 重启

## 方法三：使用启动器

1. 打开秋叶启动器
2. 点击「版本管理」
3. 检查更新
4. 一键升级

## 注意事项

| 事项 | 说明 |
|------|------|
| 备份工作流 | 更新可能影响兼容性 |
| 检查节点兼容性 | 旧节点可能不支持新版本 |
| 关注更新日志 | 了解变更内容 |

## 版本回退

### 查找之前的版本

```bash
git log
git checkout <commit-hash>
```

### 使用启动器回退

部分启动器支持版本回退功能。

## 常见问题

### Q: 更新后工作流不兼容

**A**: 某些节点需要更新到对应版本。

### Q: 更新后节点报错

**A**: 可能是节点版本不兼容，尝试更新节点。

### Q: 更新后无法启动

**A**: 
1. 检查 Python 版本
2. 检查依赖
3. 查看错误日志
