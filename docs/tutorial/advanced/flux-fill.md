# Flux Fill 修复教程

学习使用 Flux Fill 模型进行图像修复和局部重绘。

## Flux Fill 介绍

Flux Fill 是 Flux 系列的修复模型，专门用于图像编辑和局部重绘。

### 模型特点

| 特点 | 说明 |
|------|------|
| 高精度编辑 | 精确控制编辑区域 |
| 结构保持 | 保持非编辑区域 |
| 风格一致 | 整体风格统一 |

### 可用模型

| 模型 | 说明 |
|------|------|
| flux1-fill-dev | 开发版 |
| flux1-dev | 通用模型 |

## 使用步骤

### 第一步：准备图像和遮罩

1. 加载原图
2. 创建或导入遮罩
3. 确保遮罩正确标识编辑区域

### 第二步：编码

1. 使用 VAE Encode 编码原图
2. 使用 Mask 节点处理遮罩

### 第三步：设置提示词

```
# 描述要生成的内容
正向提示词：a beautiful flower garden, vibrant colors
负向提示词：blurry, low quality, distorted

# 重要提示
- 描述你想要在遮罩区域生成的内容
- 使用"in the style of"保持风格一致
```

### 第四步：采样

| 参数 | 推荐值 |
|------|--------|
| Steps | 20-30 |
| CFG | 3.5-7 |
| Denoise | 0.6-0.9 |

### 第五步：解码

使用 VAE Decode 输出最终图像。

## Mask 创建方法

### 方法一：绘图工具

使用 ComfyUI 的 Mask 节点：

```
1. 创建空白 Mask
2. 使用 Mask Draw 绘制编辑区域
3. 调整笔刷大小和硬度
```

### 方法二：图像编辑

1. 在其他软件中创建黑白遮罩
2. 白色表示编辑区域
3. 导入遮罩图像

### 方法三：自动检测

使用 Segment 或检测节点：

```
1. 使用 SAM 检测
2. 选择目标区域
3. 生成遮罩

# 适合人物、产品等自动检测场景
```

## 高级技巧

### 羽化遮罩边缘

避免生硬边缘：

```
1. 使用 Mask Blur 模糊遮罩边缘
2. blur_amount: 10-30 像素
3. 获得更自然的过渡
```

### 分层编辑

多次编辑不同区域：

```
1. 第一次编辑：整体风格
2. 第二次编辑：细节调整
3. 保持上下文连贯
```

### 结合 ControlNet

增强结构控制：

```
1. 使用 Canny 检测原图结构
2. 应用 ControlNet 控制生成
3. 保持原有构图
```

## 参数优化

### Denoise 调整

| 目标 | Denoise |
|------|---------|
| 轻微调整 | 0.3-0.5 |
| 中等修改 | 0.5-0.7 |
| 大幅改变 | 0.7-0.9 |

### CFG 调整

Flux 对 CFG 敏感，建议：

| 场景 | CFG |
|------|-----|
| 忠实原图 | 2-3.5 |
| 平衡 | 3.5-5 |
| 创意发挥 | 5-7 |

## 常见问题

### Q: 编辑区域有接缝

**A**: 
- 使用 Mask Blur 羽化边缘
- 提高 Denoise 值
- 多次微调

### Q: 生成内容不匹配

**A**:
- 检查提示词是否清晰
- 调整 CFG 值
- 确保遮罩正确

### Q: 整体风格不一致

**A**:
- 在提示词中加入风格描述
- 参考原图风格
- 使用较短的编辑范围
